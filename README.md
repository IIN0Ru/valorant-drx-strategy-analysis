# valorant-drx-strategy-analysis
# DRX 전략 분석 (VALORANT)
맵, 조합, 밴픽 데이터를 기반으로 DRX의 전략적 특성과 성과를 분석한다.

## 1. 프로젝트 개요
본 프로젝트는 발로란트 프로팀 DRX의 경기 데이터를 활용하여
맵별 승률, 조합 구성, 밴픽 전략이 실제 경기 성과로 이어졌는지를 분석하는 것을 목표로 한다.

발로란트에서 맵은 전략의 핵심 요소이므로,
맵 단위 분석을 중심으로 팀의 강점과 약점을 데이터로 정리한다.

## 2. 분석 질문
- DRX는 특정 맵에서 일관되게 높은 성과를 보이는가?
- 맵별 조합 구성에는 공통적인 패턴이 존재하는가?

## 3. 데이터 수집
- 대상 게임: VALORANT
- 대상 팀: DRX
- 데이터 범위: 공식 대회 경기
- 데이터 출처: 공개 경기 기록 웹사이트
- 수집 방법: 웹 크롤링 및 수집 자동화

## 4. 전처리 및 변수 설계
- 맵 단위 경기 데이터 구조화
- 조합을 역할군 및 전략 유형으로 단순화
- 밴픽 여부 및 승패 변수 생성

## 5. 분석 방법
- 맵별 승률 비교
- 밴픽 선택 빈도와 성과 비교
- 조합 유형별 승률 분석

## 6. 결과 및 인사이트
- DRX가 강점을 보이는 맵과 약점을 보이는 맵을 구분
- 전략적 선택과 실제 성과 간의 차이 확인

## 7. 한계 및 향후 개선
- 선수 개인 퍼포먼스 미반영
- 메타 변화에 따른 전략 변화 미분리
- 타 팀과의 전략 비교 분석 확장 가능
